Ataque/Técnica Concepto Función Ejemplos
Ingeniería Social Táctica que explota la psicología humana para manipular a las víctimas. Obtener información confidencial sin usar métodos técnicos. - Hacerse pasar por soporte técnico para pedir credenciales.
Phishing Mensajes falsos que imitan entidades legítimas. Engañar al usuario para que revele datos o descargue malware. - Email falso del banco pidiendo "actualizar la contraseña".
Pretexting Creación de una historia falsa para ganar confianza. Extraer información con un escenario falso. - Fingir ser un auditor interno para pedir datos financieros.
ARP Spoofing Manipulación de la tabla ARP. Interceptar o redirigir tráfico (Man-in-the-Middle). - Robar credenciales en una red Wi-Fi pública.
DNS Spoofing Falsificación de respuestas DNS. Redirigir tráfico a sitios fraudulentos. - Redireccionar "facebook.com" a un sitio clon.
IP Spoofing Falsificación de dirección IP de origen. Ocultar identidad o suplantar dispositivos. - Ataque DDoS con IPs falsas para saturar un servidor.
Email Spoofing Falsificación del remitente de un correo. Engañar al receptor con un email falso. - Correo que parece del jefe pidiendo dinero.
Pharming Redirección silenciosa a sitios falsos. Capturar datos mediante páginas clonadas. - Router modificado para enviar a sitios falsos.
Tailgating Entrar físicamente siguiendo a alguien. Acceder a zonas restringidas sin credencial. - Seguir a un empleado para entrar a oficinas.
Dumpster Diving Buscar información en la basura. Obtener datos físicos descartados. - Revisar papeles con contraseñas o facturas.
